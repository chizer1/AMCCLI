# AMC CLI

Solving a very niche problem for myself and maybe some others too!

Easily select multiple AMC theatres near you for a chosen date, seamlessly combine the showtimes from all selected locations into a single, convenient view, and automatically fetch detailed IMDb information for each movieâ€”saving you the hassle of looking up ratings and descriptions manually, so you can make an informed decision about which movie to watch.

## How to run

1. Go to the [Releases page](https://github.com/chizer1/AMCCLI/releases).
2. Pick a version.
3. Download the zip file that matches your operating system.
4. Unzip and run the executable (`.exe` on Windows, or the appropriate binary on other OSes).

Enjoy picking your movies with ease!
